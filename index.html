Filter Data By City Name

Type in the name of a city you would like to see weather statistics for. You can type in just the first letter(s) of the city name to see all weather data for cities with names that start with the same letter(s). The letters you enter are case sensitive, and you must click the "Filter Results" button to filter. If the data displayed does not change, there are no city names that start with the letter(s) you entered.
